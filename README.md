# Midnight English

一个文艺治愈风英语内容流微信小程序。

## 体验

像刷小红书一样刷英语。

## 技术栈

- 微信原生小程序
- 微信云开发
- Vant Weapp
- JavaScript / SCSS

## 目录结构

```
midnight-english/
├── pages/              # 页面
│   ├── index/          # 首页
│   ├── favorites/      # 收藏
│   ├── words/          # 生词本
│   └── profile/        # 我的
├── components/         # 公共组件
├── custom-tab-bar/     # 自定义 TabBar
├── styles/             # 全局样式
├── utils/              # 工具函数
└── miniprogram_npm/    # npm 构建产物
```

## 开发

1. 微信开发者工具打开项目
2. 工具 → 构建 npm
3. 编译运行
