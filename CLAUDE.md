# 项目名称

Midnight English

# 项目定位

一个文艺治愈风英语内容流微信小程序。

核心体验：
像刷小红书一样刷英语。

强调：
- 沉浸感
- 文艺感
- 轻学习
- 极简克制UI

禁止：
- 学习工具风格
- 复杂功能堆积
- 社交社区化

---

# 技术栈

- 微信原生小程序
- 微信云开发
- Vant Weapp
- 原生 JavaScript
- SCSS

禁止：
- TypeScript
- Taro
- uni-app
- React

---

# UI风格

风格：
MUJI + 深夜便利店 + 小红书轻文艺

关键词：
- 留白
- 米白
- 莫兰迪灰蓝
- 微阴影
- 呼吸感
- 柔和动画

避免：
- 高饱和
- 重阴影
- 花哨动画
- 复杂边框

---

# 页面结构

TabBar:
- 首页 Home
- 收藏 Favorites
- 生词本 Words
- 我的 Profile

---

# 首页要求

首页是核心。

功能：
- 卡片流
- 上下滑动
- 卡片默认折叠
- 点击展开全文
- 点击句子显示翻译
- 长按单词查词
- 右滑收藏
- TTS朗读

禁止：
- 真正无限swiper
- 重型动画

---

# 动画规则

动画必须轻量。

允许：
- opacity
- scale
- translateY

禁止：
- 贝塞尔轨迹
- 复杂3D动画
- 多层嵌套动画

---

# 数据结构

所有文章必须句子化存储：

{
  title,
  coverQuote,
  mood,
  scene,
  difficulty,
  readingTime,
  sentences: [
    {
      en,
      zh,
      tokens
    }
  ]
}

---

# 代码规范

- 一个页面只做一个职责
- 所有组件必须可复用
- 不允许单文件超过500行
- SCSS模块化
- 禁止内联style
- 禁止硬编码颜色

---

# 输出要求

生成代码时：
- 先说明文件结构
- 再输出代码
- 不省略关键代码
- 不使用伪代码
- 所有路径必须正确

修改代码时：
- 明确说明修改文件
- 不重复输出未修改文件
